# Matrix Rotation


Given a matrix`m`and a number`n`, return the matrix
  after rotating it 90 degrees clockwise`n`times.



**Constraints:**



  
- 1 ≤ rows, cols ≤ 10
  
- 0 ≤ n ≤ 100
  
- Matrix elements are integers




**Example:**


`f([[1, 2], [3, 4]], 1)`returns`[[3, 1], [4, 2]]`


`f([[1, 2, 3]], 2)`returns`[[3, 2, 1]]`
